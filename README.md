# 🍔 Food App
Food App is a modern and responsive food ordering web application built with React. It allows users to browse meals, view details, and manage their cart with a clean and interactive user experience.

---

## ✨ Features

- 🍽️ Browse available meals
- 🛒 Add and remove items from cart
- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚡ Fast performance using modern React practices
- 🎨 Clean and modern user interface
- 🧠 State management using React Context API
- 🚀 Deployed using Vercel
- 🧩 Component-based architecture

## 🚀 Getting Started

Follow these steps to run the project locally:

1️⃣ Clone the repository
```
git clone https://github.com/Mostafa-eskander/Food-App.git
```
2️⃣ Navigate to project folder
```
cd Food-App
```
3️⃣ Install dependencies
```
npm install
```
4️⃣ Run the development server
```
npm run dev
```
The app will run on:
```
http://localhost:3000
```

---

### 🌐 Deployment

This project is deployed using Vercel.
To deploy your own version:
```
npm run build
```
Then upload the Next.js folder to Vercel.
